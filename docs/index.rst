.. _Overview:

Overview
--------

LEAP (LEAP Estimates Animal Pose), is a framework for animal body part position estimation via deep learning.

.. _Installation:

Installation
------------

LEAP is compatible with python versions >= 3.6, installation via pip

::

    pip install leap-pose
